# Programação Orientada a Objetos | Java e desenvolvimento da API

Este repositório apresenta a construção integral de uma API desenvolvida para consolidar os fundamentos de Programação Orientada a Objetos (POO) e práticas de desenvolvimento backend em Java. O projeto foca na aplicação de conceitos teóricos em um cenário real de serviços web.

## O que foi desenvolvido?
O desenvolvimento contemplou a criação do projeto desde a sua concepção básica até a estruturação de uma API funcional para inserção e manipulação de dados. A arquitetura foi desenhada para garantir que as requisições interajam diretamente com as regras de negócio, refletindo o fluxo padrão de aplicações corporativas.

## Tecnologias e Conceitos Aplicados
A construção da API seguiu padrões de mercado para garantir um código limpo, modular e escalável. As principais competências trabalhadas incluem:

* **Gerenciamento de Dependências e Build:** Configuração estrutural do projeto utilizando Maven (`pom.xml`) para a gestão eficiente de bibliotecas e automação do build.
* **Construção de API:** Desenvolvimento completo do fluxo de dados, desde o recebimento da requisição até o processamento e inserção das informações no sistema.
* **Arquitetura de Software:** Divisão clara de responsabilidades entre as camadas da aplicação (como controladores, serviços e modelos de domínio), facilitando a manutenção e a evolução do código.
* **Pilares da POO:** Aplicação de Encapsulamento, Herança, Polimorfismo e Abstração para garantir a integridade dos dados e a reutilização de código.
* **Gerenciamento de Memória:** Uso avançado de Java Collections para armazenar, buscar e gerenciar os objetos dinamicamente durante o tempo de execução.
